# GitHubUsersApp

## Description

get a list of github API [link](https://api.github.com) users and display information about the user

---

## Tools

- **MVP**<br/>
- **Moxy**<br/>
- **Cicerone**<br/>
- **RxJava3**<br/>
- **Dagger2**<br/>
- **ROOM**<br/>
- **Retrofit**<br/>
- **ViewBinding**<br/>
